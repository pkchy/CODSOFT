# ✅ Task 1: To-Do List Application

This is a simple command-line To-Do List app made using Python.

## 📌 Features

- Add new tasks
- List all tasks
- Update existing tasks
- Delete tasks
- Store data using a JSON file

## ▶️ How to Run

1. Open terminal in this folder  
2. Run the app using:

```bash
python todo.py
